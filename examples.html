who uses it?

  • fliplog: fluent logging with verbose insight, colors, tables, emoji, filtering, spinners, progress bars, timestamps, capturing, stack traces, tracking, presets, & more...
  • lego-api renders the infinitely nestable permutations of conditionals for fuse-box so only what you use is included in the bundled api
  • d-l-l makes your webpack build faster, in just a few lines, without having to waste time with the tedious manual configuration steps required to use the DLLPlugin
  • webpack-split-plugin easily split bundles into multiple pieces with custom specifications
  • bench-chain benchmark recording - averages & graphs.
  • cli-chain easy, powerful, interactive, infinitely nestable fluent cli builder.
  • awesome-fluents more awesome fluents/chains.
  • funwithflags parse argument options. minimist in es6.
  • chain-able-find globby-compatible api, faster, & fluent.
  • chain-able-md markdown builder with a fluent api, used to create chain-able docs.
  • frisbee-chain fetch api enhancer with Frisbee, as-a-chain.
